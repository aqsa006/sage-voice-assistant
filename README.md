# Sage Voice Assistant
Sage is a Python-based voice assistant built from scratch.
It is a learning project to explore Python, automation, APIs, and AI development.

## Features
- Speech Recognition
- Text To Speech
- Open Google
- Open YouTube
- Open Calculator
- Open Spotify
- Open Clock
- Tell Current Date

## Tech Stack
- Python
- SpeechRecognition
- pyttsx3
- webbrowser
- os

## Future Plans
- Google Search
- Note taking
- Battery Monitoring
- Email Automation
- AI Integration

## status
Current Version: v0.1
Sage is actively under development and new features are being added regularly.
